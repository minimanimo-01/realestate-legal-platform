🔥 해결! Vercel이 package.json을 찾지 못하는 오류!
================================================================================

오류 메시지:
npm error path /vercel/path0/package.json
npm error enoent Could not read package.json

💡 원인: Vercel의 Root Directory 설정이 잘못됨!

================================================================================

💡 무엇이 문제였나요?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Root Directory = "./"
→ Vercel이 "./" 폴더 안에서 package.json을 찾음
→ 하지만 package.json은 루트에 있음
→ 찾을 수 없음! ❌

Root Directory = "" (비어있음) 또는 "."
→ Vercel이 루트에서 package.json을 찾음
→ 찾음! ✅
