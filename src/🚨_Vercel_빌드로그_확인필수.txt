│ 오류 3: "ReferenceError: process is not defined"                           │

│ 해결: Settings → Environment Variables 확인                                │
└────────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────┐
│ 오류 4: "Failed to resolve entry for package"                              │
├────────────────────────────────────────────────────────────────────────────┤
│ 원인: Vite 설정 또는 import 경로 문제                                       │

└────────────────────────────────────────────────────────────────────────────┘
