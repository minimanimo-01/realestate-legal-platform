1. Deployments → 실패한 배포 클릭
2. "Building" 섹션 펼치기
3. 빨간색 오류 메시지 확인

일반적인 오류:

❌ "Module not found"
   → 패키지 설치 실패
   → Install Command를 "npm install --legacy-peer-deps"로 변경

❌ "TypeScript error"
   → 타입 오류
   → Build Command를 "vite build"로 변경 (임시)

❌ "Environment variable not defined"
   → 환경변수 누락
   → 위의 4단계 다시 확인
