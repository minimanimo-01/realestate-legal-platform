# 💻 터미널(명령 프롬프트) 실행 방법

## 🎯 중요: 프로젝트 폴더에서 실행하세요!

터미널은 **다운로드한 프로젝트 폴더 안**에서 실행해야 합니다.

---

## 🪟 Windows 사용자

### 방법 1: 폴더에서 바로 실행 (가장 쉬움!) ⭐

1. **파일 탐색기**에서 다운로드한 프로젝트 폴더 열기
2. 폴더 안에서 **주소창 클릭** (폴더 경로가 표시된 곳)
3. `cmd` 입력하고 **Enter**
4. 명령 프롬프트가 해당 폴더에서 열립니다!

```
예시:
C:\Users\사용자명\Downloads\realestate-legal-platform>
```

---

### 방법 2: 우클릭 메뉴 사용

1. 프로젝트 폴더 열기
2. 폴더 안 빈 공간에서 **마우스 우클릭**
3. **"Git Bash Here"** 선택 (Git이 설치되어 있는 경우)
   
   또는
   
   **Shift + 우클릭** → **"여기서 PowerShell 창 열기"** 또는 **"명령 프롬프트 열기"**

---

### 방법 3: 명령 프롬프트에서 직접 이동

1. **Win + R** 누르기
2. `cmd` 입력하고 **Enter**
3. 프로젝트 폴더로 이동:

```bash
# 예시: D 드라이브에 있는 경우
D:
cd D:\프로젝트\realestate-legal-platform

# 예시: 다운로드 폴더에 있는 경우
cd C:\Users\사용자명\Downloads\realestate-legal-platform
```

---

## 🍎 Mac 사용자

### 방법 1: Finder에서 바로 실행 (가장 쉬움!) ⭐

1. **Finder**에서 프로젝트 폴더 열기
2. **Finder 메뉴** → **서비스** → **폴더에서 새로운 터미널 열기**

---

### 방법 2: 드래그 앤 드롭

1. **Spotlight** 검색 (**Cmd + Space**)
2. `터미널` 입력하고 **Enter**
3. Finder에서 프로젝트 폴더를 터미널 창으로 **드래그**
4. `cd ` (cd 뒤에 공백) 입력 후 폴더 경로가 자동 입력됨
5. **Enter**

---

### 방법 3: 터미널에서 직접 이동

1. **Spotlight** 검색 (**Cmd + Space**)
2. `터미널` 입력하고 **Enter**
3. 프로젝트 폴더로 이동:

```bash
cd ~/Downloads/realestate-legal-platform
```

---

## 🐧 Linux 사용자

### 방법 1: 폴더에서 바로 실행 (가장 쉬움!) ⭐

1. 파일 관리자에서 프로젝트 폴더 열기
2. 폴더 안 빈 공간에서 **마우스 우클릭**
3. **"Open in Terminal"** 또는 **"터미널에서 열기"** 선택

---

### 방법 2: 터미널에서 직접 이동

1. **Ctrl + Alt + T** (터미널 단축키)
2. 프로젝트 폴더로 이동:

```bash
cd ~/Downloads/realestate-legal-platform
```

## 🎯 Git이 설치되어 있는지 확인

터미널에서 입력:

```bash
git --version
```

**결과 예시:**
```
git version 2.40.0
```

### Git이 설치되지 않았다면?

**Windows:**
- https://git-scm.com/download/win
- 설치 후 컴퓨터 재시작

**Mac:**
```bash
xcode-select --install
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt-get install git
```

## 🆘 자주 묻는 질문

### Q: "git을 찾을 수 없습니다" 오류
**A:** Git을 설치하세요: https://git-scm.com/downloads

### Q: 프로젝트 폴더가 어디에 있나요?
**A:** 다운로드한 위치를 확인하세요:
- 보통 `다운로드` 또는 `Downloads` 폴더
- 압축 파일이라면 먼저 압축을 풀어야 합니다

### Q: 명령어를 입력했는데 아무 반응이 없어요
**A:** Enter를 눌러야 실행됩니다!

### Q: 경로에 공백이 있으면?
**A:** 따옴표로 감싸세요:
```bash
cd "C:\Users\내 이름\프로젝트 폴더"
```
